GIT Homework 1

 1. Создать внешний репозиторий c названием TXT. === open github.com (account already created) - click the "New" button - populate the "Repository name" - click the "Create repository" button  
 2. Клонировать репозиторий TXT на локальный компьютер. === git clone repository link
 3. Внутри локального TXT создать файл “new.txt”. === cat > new.txt or touch new.txt
 4. Добавить файл под гит. === git add new.txt
 5. Закоммитить файл. === git commit -m "add txt file"
 6. Отправить файл на внешний GitHub репозиторий. === git push
 7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT. === vim new.txt
 8. Отправить изменения на внешний репозиторий. === git commit -am "modify file" then  git push
 9. Создать файл preferences.txt === cat > preferences.txt or touch preferences.txt
 10. В файл preferences.txt добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT. === vim preferences.txt
 11. Создать файл sklls.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT === cat > skills.txt or touch skills.txt
 12. Отправить сразу 2 файла на внешний репозиторий. === git add files name then git commit -m "comment" then git push
 13. На веб интерфейсе создать файл bug_report.txt. === open github - open repository - Add file with the name bug_report.txt
 14. Сделать Commit changes (сохранить) изменения на веб интерфейсе. === click the "Commit new file" button
 15. На веб интерфейсе модифицировать файл bug_report.txt, добавить баг репорт в формате TXT. === open the 'bug_report.txt' file and click the 'Edit' option
 16. Сделать Commit changes (сохранить) изменения на веб интерфейсе. === click the 'Commit changes' button
 17. Синхронизировать внешний и локальный репозиторий TXT === open terminal and write git pull
